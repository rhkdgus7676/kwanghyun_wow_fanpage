Philip UI

1. 제가 직접 개발한 asMOD 애드온들과 DBM Skada등 필수 애드온을 포함한 모음집 입니다.
2. 모든 직업을 지원하고 별도의 추가 설정없이 Sample영상과 같은 구성이 가능합니다.
3. asMOD 애드온들은 최대한 기본 UI를 큰 변경 없이 와우 플레이에 도움이 되도록 강화해 줍니다.
4. asMOD 애드온들은 게임을 느려지게 하지 않습니다.
5. 별도의 추가 애드온 없이 와우 던전/레이드/전장 플레이가 가능합니다.

download philip ui --LINK--



EUI UI

1: ElvUI 원본에서 필요한 기능을 추가한 통합 애드온입니다.
2: 다른 통합 애드온 처럼 필요없는 애드온 삭제 하는 방식이 아니라,
설정창에서 [활성화 / 비 활성화] 방식입니다.
3: ElvUI 기타 애드온 EUI 사용자 사용가능 하지만, EUI 기타 애드온은 ElvUI 사용이 불가능합니다.
4: WeakAuras 버전은 게임상으로만 배포합니다.

download eui ui --LINK--
